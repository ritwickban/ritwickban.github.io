Speed Profile for Non-Holonomic Autonomous Vehicle

Pioneered a dynamic speed profile algorithm that intelligently
adjusts vehicle speed in real-time based on path curvature.
Deployed successfully in ROS, C++, and Eigen, this innovation enhances
safety standards, ensuring optimal performance and adaptability by
prioritizing safety through curvature-aware speed adjustments.

Graph-Based Planner for Warehouse Navigation

Developed a graph-based planner that optimizes warehouse navigation
by minimizing travel time and maximizing efficiency. This algorithm
is implemented in Python and ROS, and is capable of handling
complex warehouse layouts with multiple obstacles and dynamic environments
without requiring any prior map information.

Navigation Stack for Outdoor Autonomous Tractor

Designed and implemented a navigation stack for an outdoor autonomous
tractor using C++, ROS. This stack includes a global planner, local planner,
and obstacle avoidance algorithm, all of which are integrated with
control systems to ensure safe and efficient operation in outdoor environments.
